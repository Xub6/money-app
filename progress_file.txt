`C:\Users\sshuser\money-app\build\app\outputs\flutter-apk\app-release.apk`

```
cd ~/money-app && git push origin main
```
（SSH，無需 PAT）

### 圖示重新設計方式
1. 修改 `assets/icon.svg`
2. 執行：`cd /tmp && node -e "const sharp=require('sharp'); const fs=require('fs'); const svg=fs.readFileSync('C:/Users/sshuser/money-app/assets/icon.svg'); const res='C:/Users/sshuser/money-app/android/app/src/main/res/'; [[192,res+'mipmap-xxxhdpi/ic_launcher.png'],[144,res+'mipmap-xxhdpi/ic_launcher.png'],[96,res+'mipmap-xhdpi/ic_launcher.png'],[72,res+'mipmap-hdpi/ic_launcher.png'],[48,res+'mipmap-mdpi/ic_launcher.png']].forEach(([s,p])=>sharp(svg).resize(s,s).png().toFile(p))"`
